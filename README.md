🚗 DriveKhata

A full-stack Car Rental Management System designed for vehicle owners to track rentals, manage documents, and monitor payments efficiently.

✨ Features
📌 Add, update, and manage car rental records
🪪 Upload and store customer documents (Aadhar & Driving License)
🔐 Secure authentication using JWT
📊 Generate and export reports in Excel format
📱 Fully responsive UI for all devices
⚡ Progressive Web App (PWA) – installable on mobile & desktop

🚀 Getting Started
Clone the repository
git clone <your-repo-link>
cd DriveKhata
Install & run frontend
cd client
npm install
npm run dev
Install & run backend
cd server
npm install
node index.js
