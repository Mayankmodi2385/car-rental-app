# DriveKhata

A full-stack Car Rental Management System with document upload, authentication, and analytics.

## Features
- Add & manage car rental entries
- Upload Aadhar & Driving License
- JWT Authentication
- Excel report export
- Responsive UI
- Progressive Web App (Installable)

## Tech Stack
Frontend:
- React
- Axios
- CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

File Storage:
- Cloudinary
- Multer

Deployment:
- Vercel (Frontend)
- Render (Backend)

## Installation

```bash
git clone <your-repo-link>
cd project

# frontend
cd client
npm install
npm run dev

# backend
cd server
npm install
node index.js
